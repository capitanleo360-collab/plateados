# 🍽️ Plateados

## ✨ Descripción del Proyecto
Plateados es una plataforma web para la gestión de pedidos y órdenes de un restaurante. Permite a los comensales explorar el menú digital, seleccionar sus platillos favoritos y realizar su pedido de forma rápida y sencilla desde cualquier dispositivo.

## 💎 Características Principales
* **Menú Digital:** Visualización de platillos organizados por categorías (desayuno, almuerzo, postre, cena y especial).
* **Gestión de Pedidos:** modificar cantidades y realizar la orden con el monto total a pagar.
* **Autenticación de Usuarios:** Registro e inicio de sesión seguro para los clientes.
* **Panel Administrativo:** Gestión total de platillos, disponibilidad de inventario, historial de órdenes y clientes.
* **Buscador y Filtros:** Filtro rápido por tipo de comida

### 🏗️ En Desarrollo
* **Notificaciones en Tiempo Real:** Avisos por WhatsApp o correo sobre el estado de la preparación del pedido.
* **Sistema de pedido:** Módulo para pedir platillos directamente desde la web.

## 🛠️ Tecnologías Utilizadas
El proyecto está construido utilizando un stack potente y escalable:
1. **HTML5 & CSS:** Estructura y diseño.
2. **Python 3.13:** Lenguaje de programación principal.
3. **Django 6.0.1:** Framework web para la lógica de negocio y seguridad.
4. **MySQL 2.2.7:** Base de datos robusta para el manejo de órdenes, menú y usuarios.
